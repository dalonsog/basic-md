MyStyle
Definición
Los loaders se crearán utilizando la librería MyLoader. No es necesario añadir ningún estilo ya que la librería viene preconfigurada con la estética a utilizar. No obstante, en caso de que sea necesario cambiar algún estilo, se hará pasándole un objeto con los estilos al constructor, NUNCA modificando estilos en un CSS.

HTML Code
No necesario. Simplemente, el elemento que lo contendrá.
JavaScript Code
// Crear un loader pasándole el contenedor
var loader = new MyLoader($('.mybox-content'));
$('#trigger_loader_button').on('click', function(){
	// Mostrar loader (ocultará el resto del contenido)
	loader.show();
	// Solo ejemplo. El cambio de contenido 
	// se controlará en una función aparte.
	setTimeout(function(){
		$('#content1, #content2').toggle();
		// Ocultar loader (mostrará el resto del contenido)
		loader.hide();
	},2000);
});
